## 개요
세상은 확률로 돌아간다. 
확률을 지배해 던전을 평정한다.

---
## 기능 소개

### 플레이어 저장
 - Player와 Inventory 객체를 저장하는 SaveFileDTO를 만들어 XML 파일로 저장.

### 행운 로직
 - 선택지를 고를 때 마다 플레이어의 행운 스탯이 반영되는 확률성 이벤트 구현
